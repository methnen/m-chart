# Example Charts

Downloadable WordPress export files containing example charts are available for both Chart.js and Highcharts. These can be imported into any WordPress site using the standard WordPress Importer plugin (`Tools → Import → WordPress`).

The example files cover most supported chart types with sample data, making them a good starting point for understanding data formats and available settings.

- [Download Chart.js examples](/m-chart/examples/m-chart-example-charts-chartjs.zip)
- [Download Highcharts examples](/m-chart/examples/m-chart-example-charts-highcharts.zip)
