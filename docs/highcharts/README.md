# Highcharts Guide

The [M Chart Highcharts Library](https://github.com/methnen/m-chart-highcharts-library) plugin extends M Chart to support [Highcharts](https://www.highcharts.com) as the charting library.

::: warning Highcharts License
Highcharts requires a commercial license for most websites. Review [Highcharts licensing](https://www.highcharts.com/blog/products/highcharts/) before using it in production.
:::

## Setup

1. Install and activate the **M Chart Highcharts Library** plugin
2. Go to **Settings → M Chart** and select **Highcharts** as the active library
